🔧 TURBÓ SZERVIZ KEZELŐ - DESKTOP ALKALMAZÁS
===============================================

📦 TELEPÍTÉS:
- Nincs telepítés szükséges!
- Egyszerűen dupla klikk: "Turbo-Szerviz-Portable.exe"

💾 FUTTATÁS:
1. Indítsd el a "Turbo-Szerviz-Portable.exe" fájlt
2. Az alkalmazás automatikusan elindítja a helyi adatbázist
3. Megnyílik a Turbó Szerviz kezelő felület

✨ FUNKCIÓK:
✅ Teljes offline működés (nincs internet szükséges)
✅ Ügyfelek kezelése
✅ Munkalapok létrehozása és kezelése
✅ Turbó kódok és alkatrészek nyilvántartása
✅ Raktárkezelés
✅ PDF/HTML jelentések generálása
✅ Helyi SQLite adatbázis

💾 ADATOK HELYE:
- Windows: %APPDATA%\turbo-service-desktop\turbo-service.db
- Automatikus adatmentés minden használat után

🚨 ELSŐ INDÍTÁS:
- Windows biztonsági figyelmeztetés lehetséges
- Kattints "További információ" → "Futtatás mindenképpen"
- Ez normális, mert a fájl nincs digitálisan aláírva

📞 TÁMOGATÁS:
- Ez egy teljes körű turbófeltöltő szerviz kezelő
- Minden funkció ugyanaz mint a web verzióban
- Offline adatbázissal lokálisan tárolva

🎉 HASZNÁLAT:
- Új ügyfelek hozzáadása
- Munkalapok készítése turbó javítási munkákhoz
- Alkatrészek és készletek kezelése
- Jelentések nyomtatása
